# Ludex — Análisis Competitivo
## GameFi + Financial Literacy | Marzo 2026

## Posicionamiento Único

**Nadie combina las 5 cosas que Ludex tiene:**
1. AI NPCs adaptativos (Claude)
2. RPG gameplay real (no quizzes)
3. On-chain credentials (soulbound tokens)
4. Currículo de financial literacy (6 categorías, 23 lecciones)
5. Foco en LATAM (200M jóvenes)

> Cada competidor le falta al menos 3 de estas 5.

## Qué Necesitamos para $2M Seed

1. **5,000+ beta users** con 30%+ DAU/MAU ratio
2. **1 piloto escolar** firmado (incluso gratis)
3. **Carta de interés** del BID Lab o ministerio de educación
4. **Video demo** del AI NPC enseñando genuinamente
5. **Equipo**: 1 gaming veteran + 1 education expert + 1 technical founder
6. **Unit economics**: LTV:CAC >10:1, gross margin >75%
7. **18 meses de runway** con el seed

## Riesgos que Inversores Preguntarán

| Riesgo | Mitigación |
|--------|-----------|
| Costos de AI API a escala | Fine-tuning + cache + modelos más baratos para interacciones simples |
| Regulación de tokens en LATAM | Soulbound (no transferibles) = no securities |
| Calidad del juego vs AAA | MVP focused, no competimos con AAA sino con apps educativas |
| Ciclo de venta institucional | Piloto gratis → caso de éxito → venta |
| Velocidad de creación de contenido | AI-assisted content generation |
