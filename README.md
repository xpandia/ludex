# Ludex

**"Financial literacy shouldn't be a privilege. It should be a game everyone wins."**

---

## The Problem

Over **200 million young people across Latin America** lack access to meaningful financial education. Traditional methods fail them: boring curricula, zero engagement, no real-world incentive. Meanwhile, they spend hours every day gaming. The gap between what they need to learn and how they want to learn it is massive -- and growing.

## The Solution

**Ludex** is a Play-to-Learn GameFi platform on OneChain that turns financial literacy into an interactive quest-based experience. Players take AI-generated challenges across six financial categories, chat with four AI NPC mentors powered by Claude, earn XP and LDX tokens, collect achievement badges, and climb the leaderboard. A comprehensive Move smart contract on OneChain records credentials, manages the token economy, and tracks progress on-chain.

Learn by playing. Earn by learning. No textbooks. No lectures. Just progress.

---

## How It Works

### 1. Play
Open the Ludex app. Browse quests across 6 financial categories -- budgeting, investing, saving, credit, crypto, and taxes. Accept challenges generated by AI.

### 2. Learn
AI-generated quizzes test real financial concepts with multiple-choice questions tailored to your level. Four AI NPC mentors (Professor Luna, Trader Rex, Crypto Sage, Credit Fox) provide personalized coaching through chat. A structured curriculum of 23 lessons guides your progression.

### 3. Earn
Pass challenges to earn XP and LDX tokens. Level up, unlock badges, climb the leaderboard. Achievements are designed to be minted on OneChain as verifiable soulbound credentials.

## Tech Stack

| Layer | Technology |
|---|---|
| **Blockchain** | OneChain (Move) -- single comprehensive smart contract (`ludex_game.move`, 878 lines) covering LDX token, player profiles, quests, badges, staking, leaderboard, teams |
| **Backend** | Python / FastAPI -- 21 REST endpoints, JWT authentication, SQLite persistence (WAL mode, 8 tables), seed data |
| **Frontend** | Vanilla HTML / CSS / JS -- premium landing page (`index.html`) with animations and live backend data + 5-tab game app (`app.html`) with quests, NPC chat, leaderboard, profile. Multi-language support (English, Spanish, Traditional Chinese) with localStorage persistence. |
| **AI** | Claude API (Anthropic) -- 4 NPC mentors with adaptive dialogue, AI-generated quiz challenges, fallback responses for offline mode |

## Smart Contract (Move on OneChain)

A single comprehensive contract (`ludex_game.move`) that handles:

- **LDX Token** -- Custom fungible token with admin-held mint/burn/freeze capabilities
- **Player Profiles** -- On-chain registration, XP, leveling (up to 100), streaks
- **Quest System** -- 6 financial categories, configurable difficulty, XP/token rewards, expiration
- **Badge System** -- Soulbound achievement badges with 5 rarity tiers and eligibility checks
- **Staking** -- Lock LDX tokens for graduated reward multipliers (1.2x-2x)
- **Leaderboard** -- On-chain ranking, seasonal resets (30-day), top 100
- **Teams & Social** -- Team creation (max 5 members), friend lists
- **Pause Mechanism** -- Admin can pause player-facing operations

## How to Run

### Prerequisites

- Python 3.11+
- Anthropic API key (optional -- AI features fall back to template responses without it)

### Backend

```bash
cd src/backend
pip install -r requirements.txt
export LUDEX_ANTHROPIC_API_KEY="your-anthropic-api-key"  # optional
python -m uvicorn server:app --reload --port 8000
```

Server at `http://localhost:8000` | API docs at `http://localhost:8000/docs`

### Frontend

```bash
cd src/frontend
python3 -m http.server 3000
```

- Landing page: `http://localhost:3000/index.html`
- Game app: `http://localhost:3000/app.html`

Or simply open `src/frontend/index.html` and `src/frontend/app.html` directly in a browser (update the API base URL in the HTML if needed).

```bash

```
